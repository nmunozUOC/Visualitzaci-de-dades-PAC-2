# Visualització de dades PAC 2
Visualització de dades PAC 2 Natalia Muñoz

Aquest repositori conté tres visualitzacions diferents desenvolupades com a part de la PAC 2 de l’assignatura **Visualització de Dades (UOC)**.  
Cada visualització utilitza un dataset diferent i està implementada en **R** o **Python**, segons el cas.

## Visualitzacions
### 1. Circle Packing
- **Fitxer de codi:** 'CIRCLE_PACKING_.R'
- **Llenguatge:** R  
- **Dataset utilitzat:** Video Game Sales

  Pots descarregar-lo des de la següent URL:  
  [Descarrega el dataset aquí](https://www.kaggle.com/datasets/gregorut/videogamesales)  

### 2. Sunburst Chart
- **Fitxer de codi:** 'sunburst_ridgeline_chart.ipynb' 
- **Llenguatge:** Python  
- **Dataset utilitzat:** Superstore Dataset

  Pots descarregar-lo des de la següent URL:  
  [Descarrega el dataset aquí](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)  


### 3. Ridgeline Chart
- **Fitxer de codi:** 'sunburst_ridgeline_chart.ipynb' 
- **Llenguatge:** Python  
- **Dataset utilitzat:** Flights  
  **Nota:** Aquest dataset **no s’ha pogut pujar al repositori** per la seva mida.  

  Pots descarregar-lo des de la següent URL:  
  [Descarrega el dataset aquí](https://www.kaggle.com/datasets/matinsajadi/flights)  

## Notes addicionals

- El codi del **Circle Packing** està implementat en **R**.  
- Els codis del **Sunburst Chart** i del **Ridgeline Chart** estan implementats en **Python** dins del mateix notebook ('sunburst_ridgeline_chart.ipynb'). 
- Els datasets s’han d’ubicar dins de la carpeta 'data/'.
